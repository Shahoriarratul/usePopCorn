# popcorn

A Next.js + Tailwind CSS movie app for searching movies, viewing details, and managing a watched list.

## Scripts

- `npm run dev` - start the development server
- `npm run build` - build for production
- `npm start` - run the production server

## Notes

- The app uses the App Router in `src/app`.
- Movie data now comes from TMDB.
- Set `NEXT_PUBLIC_TMDB_API_KEY` in `.env.local` using a TMDB API key.
